This project consists of different parts,
each part is implemented separately (in ipynb format) and the order of implementation of each part is as follows:
1) Data Cleaning
2) Handeling Numerical and Categorical Data
3) Data Visualization
4) Classification
5) Regression
6) Feature Reduction Techniques
7) Ensembling Techniques
8) KFold Cross Validation and BootStrap Sampling
9) Clustering- KMeans- KMedoids and Hierarchial
10) Association Rules
